# Changelog — v1.9.0

## Novidades

### Backup mensal obrigatório
- Usuários são solicitados a realizar backup dos dados uma vez por mês
- Fluxo de backup integrado ao ciclo de uso do app

### Unicidade de conta Google
- Cada conta Google pode estar vinculada a apenas um perfil Veeda
- Validação implementada no cadastro e no login via Google

### Recuperação de conta via Google
- Usuários que perderam acesso ao perfil local podem recuperar a conta usando o Google
- Fluxo de recuperação guiado diretamente na tela de login

### Registros compartilhados completos
- Visualização completa de registros de dias compartilhados por outros usuários
- Sistema de comentários funcional em dias compartilhados
- Sincronização e deduplicação de dados aprimoradas
- Melhorias na timeline de dias recebidos
